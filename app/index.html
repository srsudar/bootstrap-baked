Bootstrap with grunt-bake!

A fully functional template of a Boostrap site. This template uses hrefs to navigate with the navbar and shows how you can use Grunt to keep all reusable elements (like the navbar) in one place.

About

Everyone seems to agree, Boostrap is awesome. It provides a lot of functionality right out of the box. One thing it does not provide, however, is a prescriptive way of doing things. On the one hand, great! Do what you want. On the other hand, if you're just starting out, you might like a little bit more hand holding. I know I wouldn't have turned it down.

This site is intended to serve as a template for using hrefs to provide navigation via the navbar. Further, the repo itself includes a sophisticated workflow using Grunt based on the setup provided by Yeoman and the webapp generator. See the README in the repo for more details.
